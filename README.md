# TicTacToe

A simple TicTacToe game built to practice writing a gem

## Installation

While this is a Gem, it's also simply a game.

To play clone this repo on to your local machine:

```
git clone git@github.com:MarchandMD/TicTacToeGemTutorial.git
```

That's it!

## Usage

```
ruby example/example_game.rb
```
